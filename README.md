# Indian Liver Patient Records
Patient records collected from North East of Andhra Pradesh, India

## Run loaclly
### Linux / macOS
```
pip install virtualenv
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```
### Windows
```
pip install virtualenv
py -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
py main.py
```
